# haoyangc2001.github.io

Static GitHub Pages site for `https://haoyangc2001.github.io/`.

## Files

- `index.html`: page structure
- `styles.css`: visual system and responsive layout
- `script.js`: editable content and lightweight animation

## What to edit first

Open `script.js` and replace:

- `location`
- `status`
- `projects`
- `links`
- `mailto:your-email@example.com`

If you want your exact full name, headline, or biography changed, edit `index.html` and `script.js`.

## Deploy

1. Create a GitHub repository named `haoyangc2001.github.io`
2. Copy these files into that repository root
3. Commit and push to the `main` branch
4. Open `https://haoyangc2001.github.io/`

## Notes

- No build step is required
- Works directly with GitHub Pages static hosting
- Google Fonts is used for typography
